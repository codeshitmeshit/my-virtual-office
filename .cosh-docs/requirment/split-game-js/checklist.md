## 核心渲染验证

### CHK-001 — 办公室环境渲染完整性
- **验证方法**：加载页面，目视检查 canvas 上所有家具、墙壁、窗户、时钟、白板、分支标识牌是否完整显示。
- **预期结果**：所有元素与拆分前完全一致，无缺失、无错位。
- **关联需求点**：拆分不影响 `drawEnvironment()` 及所有家具绘制函数。

### CHK-002 — Agent 外观渲染一致性
- **验证方法**：检查多个不同 Agent（有不同发型、服装、头饰、眼镜、道具）的外观是否与拆分前一致。
- **预期结果**：发型、服装颜色、头饰、眼镜、手持道具均正确渲染。
- **关联需求点**：Agent appearance 模块拆分后保持绘制逻辑不变。

### CHK-003 — Agent 位置与动画
- **验证方法**：观察 Agent 是否在正确的 desk 位置、移动动画是否流畅、朝向是否正确。
- **预期结果**：Agent 站在正确工位，走动动画流畅，faceDir 正确。
- **关联需求点**：Agent class、desk assignment 逻辑不变。

### CHK-004 — 天气渲染
- **验证方法**：在有雨/雪/雷暴的天气条件下检查窗口雨滴、积雪、闪电、天气粒子效果。
- **预期结果**：雨滴动画、积雪累积、闪电闪烁、粒子下落均正常。
- **关联需求点**：Weather 系统拆分后渲染行为不变。

- **关联需求点**：性能分析模块和 rim light cache 正常工作。
