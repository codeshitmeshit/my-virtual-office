- [x] CHK-002 长期项目且没有项目定时任务时展示推荐入口。
  - 验证方法：打开一个已标记长期项目且没有项目 cron 的项目详情页。
  - 预期结果：项目详情页展示创建项目定时任务的推荐提示或入口。
  - 关联需求点：长期项目推荐创建定时任务。

- [x] CHK-003 已有项目定时任务时不重复展示推荐噪音。
  - 验证方法：为长期项目创建一个项目定时任务后重新打开项目详情页。
  - 预期结果：推荐提示消失或降级，不反复要求用户创建。
  - 关联需求点：体验打磨和低噪音原则。

- [x] CHK-004 推荐入口只打开创建表单，不自动创建或启用 cron。

  - 预期结果：没有新增 cron；只有用户确认保存后才创建项目定时任务。

  - 预期结果：与项目详情页“新建定时任务”表单一致，不产生第二套创建逻辑。
  - 关联需求点：复用既有 Phase 2+3 能力。
