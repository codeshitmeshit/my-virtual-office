验证限制

本次验证没有启动真实 dev server，没有验证真实浏览器 EventSource，也没有验证真实 Codex/Claude/Hermes 认证环境。Node 检查是源码 marker/静态回归，不是运行中的 SSE 流测试。真实环境排查时，应先跑这些最小测试，再检查 provider binary/auth/API server。
