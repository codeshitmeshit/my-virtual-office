Claude Code CLI stream-json bridge

Claude Code 的 bridge 没有单独的 app-server runtime；adapter 直接启动 Claude CLI，并解析 `--output-format stream-json` 的 stdout。server run worker 再把 progress snapshot 差量转成 SSE。

入口：POST /api/claude-code/runs终止：ProviderRunBridge terminal event置信度：中高
详细链路代码摘录数据流验证概念

详细链路说明

1. run handler 创建 run

`_handle_claude_code_run_start` 校验 message 和 agent，创建 runId、progressId、events queue，并注册到 ProviderRunBridge。

2. worker 注入 `_onProgress`

run_body 带 `_onProgress`，后续 provider 每解析一段 stdout 就会回调 snapshot。

3. `_handle_claude_code_chat` 调 adapter

server 取 `ClaudeCodeProvider`，根据 agent profile、conversationId/sessionId 处理 history 和 session resume。

4. adapter 启动 CLI

`send_chat_message` 组装 `claude -p --output-format stream-json --verbose --include-partial-messages`，按需追加 `--resume`、`--agent`、`--model`、permission mode。

5. `_ClaudeStreamState` 聚合 stdout

JSONL stdout 包含 system、assistant/tool_use、stream_event delta、tool_result、result。state 聚合 reply、tools、usage、session_id。

6. server worker diff snapshot

worker 对 reply、thinking、tokenUsage、tool 状态做去重和差量判断，emit `message.delta`、`reasoning.available`、`session.metrics`、`tool.*`。

7. terminal result 输出

CLI 退出后 result 写入 ProviderRunBridge，成功发 `run.completed`，失败发 `run.failed`。

关键代码摘录

# app/providers/claude_code.py
cmd = [self.binary, "-p", "--output-format", "stream-json", "--verbose", "--include-partial-messages"]
if session_id:
    cmd.extend(["--resume", session_id])
if profile and profile not in {"main", "local"}:
    cmd.extend(["--agent", profile])

这段说明 Claude bridge 的原生协议边界就是 CLI stream-json。

# app/server.py
def on_progress(run_state):
    _publish_claude_code_progress(...)
    if reply and reply != last_reply:
        enqueue("message.delta", ...)
    for tool in run_state.get("tools") or []:
        enqueue("tool.started" / "tool.completed" / "tool.failed", ...)

server 不直接读 Claude stdout，而是消费 adapter 提供的 snapshot。

关键概念补充

Snapshot diffing：Claude adapter 回调的是当前累计状态，server 需要自己比较 last_reply、last_thinking、seen_tools，避免重复发 SSE。

ProviderPath：Claude 链路的 `providerPath` 是 `claude-code-cli`，用来区分 Codex app-server 和 Hermes API。
