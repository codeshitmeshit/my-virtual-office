风险声明与分析边界

本次逆向分析基于静态代码阅读和 Git 历史，以下事项需要注意：

  • 未运行测试：tests/ 目录存在但未在本分析中执行，无法确认测试覆盖率与通过状态。

  • chat.js 后半部分（100-2925 行）：多窗口管理器的完整实现细节未完全验证。
  • Git 初始提交前史：2b7b6ae 是仓库第一个提交（52 文件/27445 行），无法验证 pre-history。
  • Elix 与 eliautobot 的关系：Git 历史显示两个不同作者，但无法验证 eliautobot 是 AI agent 还是人类使用自动化别名。
  • 架构图为推断图：所有架构图和调用链图基于代码证据推断生成，不是项目官方文档。
