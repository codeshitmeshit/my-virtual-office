🌐 Agent Browser
Give your AI agents a real web browser they can control. You can watch them browse in real time through a live viewer panel.

How it works: Your agents use CDP (Chrome DevTools Protocol) to navigate, click, and read pages. The Viewer is a live screen you see in the office — like watching over their shoulder.

Compatible with: KasmVNC, headless Chromium, browserless.io, Playwright, or any setup that exposes CDP. The viewer can be any VNC/noVNC/web viewer.
The Chrome DevTools Protocol endpoint. Usually runs on the same machine as Virtual Office. Use 127.0.0.1 if the browser container is local.
The URL that loads in the floating browser window. Must be reachable from your browser (not localhost if you're accessing remotely). Use HTTP to avoid certificate issues.
